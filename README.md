# Weighted-Sparsity-Score-Batch-Mitigation-For-Multicohort-Microbiome-Studies
Bioinformatics pipeline predicting ASD from microbiome multicohort data. Uses a novel "Causal Sterilization" method (WSS &amp; MDS) to annihilate geographic Batch Effects and Shortcut Learning, evaluated via strict LODO validation across Traditional ML and Next-Gen architectures.
